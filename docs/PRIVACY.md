# 隐私政策

最后更新：2026年3月17日

## 我们收集什么

**搜索服务 (2345.desuwa.org)**
- 本服务不收集任何个人信息
- 搜索请求仅用于执行搜索功能，不会保存或追踪

**API 服务**
- 用户注册：仅收集邮箱地址用于账号管理
- API 使用记录：仅用于计费和限额控制（记录查询次数，不记录搜索内容）
- API Key：用于身份验证，不会与第三方共享

## 我们不收集

- 搜索历史
- IP 地址（仅用于速率限制，限制后不存储）
- 用户代理信息

## 数据处理

- 所有数据仅存储在中国境内的服务器
- 用户可随时请求删除账户及相关数据

## 第三方服务

- 本服务使用 Cloudflare 作为 CDN
- 搜索引擎数据来源于 [2345.lgbt](https://2345.lgbt) 公开收录的网站

## 联系我们

如有隐私相关问题，请通过 GitHub Issues 联系。
