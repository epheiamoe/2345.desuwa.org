隐私政策

最后更新：2026年3月17日

我们收集什么

搜索服务 (2345.desuwa.org)

  • 本服务不收集任何个人信息
  • 搜索请求仅用于执行搜索功能，不会保存或追踪

API 服务

  • 用户注册：仅收集邮箱地址用于账号管理
  • API 使用记录：仅用于计费和限额控制（记录查询次数，不记录搜索内容）
  • API Key：用于身份验证，不会与第三方共享

我们不收集

  • 搜索历史
  • IP 地址（仅用于速率限制，限制后不存储）
  • 用户代理信息

数据处理

  • 所有数据存储在美国境内的服务器
  • 我们使用 Cloudflare 作为 CDN 和安全防护，部分临时数据（如缓存、速率限制日志）可能在全球节点处理，但核心用户数据（如 API 注册邮箱）存储在美国
  • 用户可随时请求删除账户及相关数据

国际用户注意

本服务托管在美国。如果您从欧盟、英国或其他地区访问，请注意您的信息可能被传输并在美国处理，美国的数据保护法律与其他地区可能不同。本服务不专门针对欧盟/英国用户提供，也不主动向这些地区提供货物或服务。

第三方服务

  • 本服务使用 Cloudflare 作为 CDN
  • 搜索引擎数据来源于 2345.lgbt 公开收录的网站

联系我们

如有隐私相关问题，请通过 GitHub Issues 联系。
