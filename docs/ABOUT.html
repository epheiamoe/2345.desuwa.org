2345.desuwa.org

关于

2345.desuwa.org 是一个专为跨性别群体打造的开源垂直搜索引擎。

我们只收录来自 2345.lgbt 导航站的可靠资源。

如果您喜欢，请在GitHub为我们的项目点个Star

我们的核心原则

  • 只收录可靠来源：全部数据来源于 project-trans 生态下的 2345.lgbt，不主动爬取外部未知网站，避免低质、有害或过时信息。
  • 极致隐私保护：我们不记录任何搜索历史、不保存用户 IP、不使用追踪 Cookie。即使通过 TOR + NoScript 访问，大部分功能也能正常使用。
  • 开源透明：项目完全开源（LGPLv3 协议），欢迎社区参与贡献、部署自己的实例，或基于 API 开发工具。
  • 社区导向：（主要）为中文（简体/繁体）跨性别朋友提供快速、精准、可信的知识获取途径，减少在茫茫网络中翻找靠谱资料的痛苦。

技术实现

  • 前端：简洁的 PHP 页面 + 标签筛选 + 多语言支持
  • 后端搜索：Meilisearch（极速且支持中文分词）
  • 数据采集：Scrapy 爬虫（仅限已知可靠域名）
  • API：提供开放API接口，您可以自己调用或通过 Anthropic Skill 调用。
